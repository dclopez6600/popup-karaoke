ESS Restaurent is a trendy and modern HTML5 Template for Hotels & Restaurent. It comes with powerful, beautiful and high-performance web design. Built with Bootstrap Framework 5.1, HTML5, CSS3, jQuery and Javascript.

ESS Restaurent template has a fully responsive layout. It fits perfectly on various displays and resolutions from regular desktop screens to tablets, iPads, iPhones and small mobile devices.
